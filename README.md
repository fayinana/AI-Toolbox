## ğŸš€ Quick Start

### Prerequisites

- Node.js `v18+`
- npm `v9+` or yarn

### Installation

```bash
git clone https://github.com/fayinana/ai-toolbox.git
cd ai-toolbox
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Testing

```bash
npm test           # Unit tests
npm run test:cov   # Test coverage report
```

### Building for Production

```bash
npm run build
```
